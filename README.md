# CareerFlux AI Frontend

A modern frontend-only React.js + Tailwind CSS project for the CareerFlux AI platform.

## Included Modules
- User Registration and Login
- Career Goal Selection
- Career Roadmap Generator
- Skill Assessment and Exam
- Performance Analysis and Suggestion
- AI Mock Interview
- Resume Builder
- ATS Resume Score Checker
- User Dashboard

## Run Locally
```bash
npm install
npm run dev
```

## Tech Stack
- React.js
- Tailwind CSS
- React Router DOM
- Recharts
- Lucide React
